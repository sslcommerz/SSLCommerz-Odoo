Payment Provider Configuration

Enter SSLCommerz Store ID and Password.

For registration in Sandbox/Testing, click the link https://developer.sslcommerz.com/registration/
For registration in Production, click the link https://signup.sslcommerz.com/register

In Test Mode enter Sandbox credentials and set State to Test Mode.
In Live enter Production credentials and set State to Enabled.

Now customers will be able to select SSLCommerz as payment gateway

Gateway Page

Live Environment: All transactions made using this environment are counted as real transactions. URL starts with https://securepay.sslcommerz.com

Sandbox Environment: All transactions made using this environment are counted as test transactions and have no effect on accounting. URL starts with https://sandbox.sslcommerz.com
